# Hounslow 2154 — Final Day Checklist

Interactive checklist for the store closure. Saves progress automatically.

## Setup (2 minutes)

### 1. Create a GitHub repo
- Go to [github.com/new](https://github.com/new)
- Name it something like `hounslow-checklist`
- Set to **Public** (required for free GitHub Pages)
- Click **Create repository**

### 2. Upload the file
- Click **"uploading an existing file"** on the empty repo page
- Drag in `index.html`
- Click **Commit changes**

### 3. Enable GitHub Pages
- Go to **Settings** → **Pages** (left sidebar)
- Under **Source**, select **Deploy from a branch**
- Branch: **main**, folder: **/ (root)**
- Click **Save**
- Wait ~60 seconds, then your site is live at:
  ```
  https://YOUR-USERNAME.github.io/hounslow-checklist/
  ```

### 4. Share with the team
Send the URL to anyone who needs it. Each person's checklist saves independently on their own device.

## How it works

- **Tap any item** to check/uncheck it
- **Type counts** into the tally boxes as you pack
- **Progress bar** updates automatically
- **State saves** to the browser — close and reopen, it's still there
- **Reset all** button in the header clears everything

## Notes

- State is saved per device (localStorage), so each team member has their own progress
- Works offline once loaded
- Mobile-friendly — designed for phone use on the shop floor
